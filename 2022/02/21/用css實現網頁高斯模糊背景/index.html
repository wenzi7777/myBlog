用css實現網頁高斯模糊特效


Attention: 這篇文章探討的是如何實現背景高斯模糊，而不是整個視圖高斯模糊！！！


大概效果是這樣：


沒有高斯模糊


使用高斯模糊效果



怎麼樣？？是不是感覺好漂亮的效果，那麼如何實現呢(¯︶¯)


先來看看DOM結構

其實有兩種方法，這是方法一，相容性較好！



-> 外層有一個 id=animeDetails 的Div標籤，用於撐起整個頁面的高度


-> 內層使用了 class=animeDetails_mask 的一個Div標籤用於放入要展示的內容，例如現在我放入的是contents這幾個字


-> 現在開始設定背景圖及高斯模糊，用到了偽元素唷（≧∇≦）




!!!終於，完成惹，來看看效果吧



好像，真的有高斯模糊了，但是為什麼圖會不透明度這麼低，讓我怎麼看字啊！

對惹！在 Div.animeDetails_mask 上設定背景色就好了吧！

不管用欸，這是怎麼回事呢？

其實答案很簡單， ::before 使用了position: absolute，已經在文檔流最上方了，會吧後面的背景色遮住。

解決方案就是，用 ::after 設定背景色
